# 电感

## 电感
电感最重要的特性——电感上的电流不能突变。
电感能够把电能转换成电磁能（电感的本质原理是电磁感应）
电感单位：
`电感`是用于描述器件对抗电流变化的能力，如果对抗电流变化的能力越强，那么电感的感性越大，反之越小。

## 计算公式

1. 电压关系：$V(t)=-L\frac{di}{dt}$
2. 阻抗：$X_C=2\pi fL$
3. 时间常数：$\tau=\frac{L}{R}$
4. 能量：$W=\frac{Li^2}{2}$
5. 电流和时间关系：$I=\frac{V}{R}(1-e^\frac{-R\tau}{L})$
6. 电压和时间关系：$V=Ve^\frac{-R\tau}{L}$
